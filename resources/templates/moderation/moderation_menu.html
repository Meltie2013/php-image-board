Moderation Control Panel

MCP

Overview
Dashboard Monitor queue health, image totals, and storage pressure.
Image Queue
Pending Review uploaded images waiting for approval, sensitivity, or rejection. Reported Reserved for future user reports and moderation escalations. DMCA Reports Reserved for takedown processing, rights review, and documentation.
Image Tools
Comparison Measure similarity between approved images before moderation decisions. Rehashing Rebuild aHash, pHash, and dHash values for approved image records.
Comments
Pending Comments Placeholder for future comment approval and escalation review. Reported Comments Placeholder for reported discussion moderation and enforcement.
